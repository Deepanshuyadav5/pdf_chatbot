# IntelliDoc - Multi-PDF Conversational AI Chatbot

AI-powered chatbot that enables natural language conversations with multiple PDF documents using RAG architecture.

## Features
- Upload and process multiple PDFs simultaneously
- Semantic search using vector embeddings
- Context-aware conversations with memory
- Clean, intuitive Streamlit interface

## Tech Stack
- **LangChain** - RAG framework
- **Ollama** - Local embeddings (nomic-embed-text)
- **FAISS** - Vector database
- **Gemma 3** - LLM for responses
- **Streamlit** - Web interface
- **PyPDF2** - PDF processing
